Reviewer Guide

Cue includes a visible secondary Try Demo action during onboarding when signed out.

Demo mode

The demo is a curated, read-only scenario showing a premium account that is active and expiring soon. Demo labels are visible on relevant surfaces. Diagnostics preview is available. Diagnostics export is intentionally disabled. Demo mode does not trigger live notification permission prompts.

Live flow

The live product flow uses Real-Debrid's official OAuth device flow and only reads account status plus local reminder settings.

Independence

Cue is an independent app for Real-Debrid users and is not affiliated with, endorsed by, or operated by Real-Debrid.
